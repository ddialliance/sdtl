Structured Data Transformation History (SDTH) extends the W3C PROV model to allow searching and querying data management programs used for statistical data. For example, SDTH easily reconstructs the lineage of a variable and shows which commands and other variables affected it. SDTH adds data types not found in PROV, and it takes a command-level approach to programs and command scripts. SDTH is expressed in Resource Description Framework (RDF) and can be queried with tools like SPARQL.
